PyDCG

Python's Discrete and Combinatorial Geometry Library is a collection of implementations of algorithms for Combinatorial and Computational Geometry. It does not aim at comprehensive; it is rather the tools that we use in our day to day research in Combinatorial Geometry.

Please refer to the docs folder for the documentation on each module.

This project is maintained by rfabila and csaorl
